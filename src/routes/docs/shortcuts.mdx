## Tips

- The search is fuzzy - you don't need to type the full name
- Results are ranked by frequency - frequently used apps appear first
- Use `--show-actions` to reveal [Desktop Action] entries on each item
- Hook into `on_keypress(key)` in a Lua plugin to handle custom keyboard shortcuts
